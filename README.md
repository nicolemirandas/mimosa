MIMOSA

Foi desenvolvido para a aula de desenvolvimento de software onde o objetivo era utilizar as estruturas de repetição aprendidas. O projeto é inspirado no conceito de Tamagotchi, simulando um pet virtual.

SOBRE O PROJETO

Mimosa é um projeto simples e inicial em C onde o usuário interage com o animal de sua escolha, realizando ações para manter o bem-estar do seu pet. Conforme as escolhas feitas, os atributos do animal são alterados, tornando a experiência dinâmica e permitindo diferentes resultados.

OBJETIVOS DO PROJETO

Durante o desenvolvimento deste projeto foram praticados os seguintes conceitos:
- Estruturas condicionais (if, else if e else);
- Estrutura de seleção (switch-case);
- Estruturas de repetição (while e for);
- Manipulação de variáveis;
- Entrada e saída de dados (scanf e printf);
- Organização da lógica do programa;
- Criação de menus interativos.

FUNCIONALIDADE

- Menu de interação com o pet;
- Sistema de ações que alteram o estado da personagem;
- Execução contínua utilizando laços de repetição;
- Diferentes respostas de acordo com as escolhas do usuário;
- Encerramento do programa por opção do usuário ou por condições definidas na lógica.

TECNOLOGIA UTILIZADA

- Linguagem C e compilador GCC
